Whidbey Island Distillery

A color-coded label system for a small-batch distillery — and the living style guide that keeps every expression on-brand and TTB-compliant.

Whidbey Island Distillery
Role
Brand & packaging design

Overview

Whidbey Island Distillery makes small-batch spirits on Whidbey Island, Washington. Each release is a distinct product — a rye, an oak-finished expression, a berry liqueur — but they all need to read as one family on a crowded back-bar, and every one of them has to satisfy the same federal labeling rules.

The work was a label redesign that grew into a system: a color-coded product line, a reusable label architecture, and a living style guide that lets the distillery add new expressions without redrawing the label from scratch each time.

The challenge

A spirits label is one of the most constrained surfaces in design. Beyond the brand, the U.S. TTB requires specific mandatory information — class and type, alcohol content, net contents, the government warning — placed and sized to spec. Get it wrong and the label doesn't ship.

On top of that, a growing product line creates a second problem: consistency. Every new release is a chance for the system to drift — a slightly different type size here, an off-brand color there. The design had to hold the line on both fronts at once: regulatory precision and brand coherence, across a range that will keep expanding.

Approach

I treated the label as a structured document, not a one-off illustration. The mandatory fields live in a fixed architecture — a place for every required value — so compliance is built into the template rather than checked at the end.

The ornamental brand layer sits on top of that structure, giving each expression room for personality while the regulated anatomy underneath stays identical from product to product.

A color-coded product line

Rather than design each release in isolation, the line is organized by color. A customer learns the code once and can navigate the whole range — the rye, the cask-finished series, the berry liqueurs — at a glance. New expressions slot into the existing coding instead of inventing their own.

A documented system

The system is captured in a style guide the distillery can work from directly — color, type, spacing, the logo lockups, and the label components collected in one place, so the brand stays consistent as the range grows.

Outcome

The distillery has a label system it can extend on its own: a new release starts from a compliant template and an established color code, not a blank page — so the line can keep expanding without losing consistency.
